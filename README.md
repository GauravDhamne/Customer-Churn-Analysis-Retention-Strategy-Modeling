## Project Overview
This project focuses on analyzing customer behavior and predicting customer churn using Machine Learning techniques. The goal is to identify customers who are likely to leave a service and understand the key factors affecting customer retention.

The project combines data analysis, preprocessing, machine learning, SQL querying, and dashboard visualization to generate business insights and support retention strategies.

## Objectives
- Analyze customer behavior and service usage patterns
- Identify factors influencing customer churn
- Build a machine learning model for churn prediction
- Perform data preprocessing and feature engineering
- Generate insights for customer retention strategies
- Create visual reports and dashboards for business understanding

## Project Workflow

### 1. Data Collection
- Imported customer dataset for analysis
- Examined dataset structure and column information
- Checked missing values and data quality

### 2. Data Preprocessing
- Handled missing and duplicate values
- Converted categorical data into numerical format
- Performed feature scaling and transformation
- Prepared dataset for machine learning models

### 3. Exploratory Data Analysis (EDA)
- Analyzed customer demographics and behavior
- Identified churn distribution patterns
- Explored relationships between services and churn
- Visualized important trends using charts and graphs

### 4. Feature Engineering
- Selected important features affecting churn
- Created meaningful variables for model improvement
- Optimized dataset for better prediction performance

### 5. Machine Learning Modeling
- Built initial churn prediction models using Scikit-learn
- Trained and evaluated classification algorithms
- Compared model performance using evaluation metrics
- Identified important factors contributing to customer churn

### 6. Dashboard & Reporting
- Created interactive Power BI dashboards
- Visualized churn trends and customer insights
- Presented findings in a business-friendly format

## Key Insights
- Customers with lower engagement showed higher churn probability
- Service usage patterns significantly affected retention
- Certain subscription and support features influenced customer loyalty
- Data-driven insights can help improve retention strategies
